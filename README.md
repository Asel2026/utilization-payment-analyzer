# Утилизационный платеж на электромобили

Локальный аналитический веб-проект «Обоснование внедрения утилизационного платежа на электромобили в Республике Казахстан путем отмены нулевого коэффициента»

## Описание проекта

Данный проект представляет собой полностью автономный аналитический веб-проект, предназначенный для исследования, моделирования и обоснования внедрения утилизационного платежа на электромобили в Республике Казахстан.

**Основные особенности:**
- Полностью локальная работа без доступа к интернету
- Без серверной части и баз данных
- Все данные хранятся в формате JSON
- Интерактивные калькуляторы для расчетов
- Визуализация данных с помощью SVG-диаграмм
- Возможность экспорта в различные форматы

## Установка и запуск

## Калькуляторы

Проект включает три интерактивных калькулятора:

### 1. Калькулятор утилизационного платежа

Расчет размера утилизационного платежа по формуле:
```
Платеж = Базовая ставка × МРП × Коэффициент
```

Параметры:
- Тип электромобиля (определяет коэффициент)
- Значение МРП
- Базовая ставка

### 2. Финансовый прогноз

Прогноз поступлений от утилизационного платежа на заданный период.

Параметры:
- Начальный парк электромобилей
- Годовой рост парка
- Средний платеж
- Период прогноза

### 3. Прогноз рынка

Прогноз развития рынка электромобилей по различным сценариям.

Параметры:
- Текущий парк электромобилей
- Сценарий развития (оптимистичный/базовый/пессимистичный)
- Горизонт прогноза

# PRD — Локальный аналитический веб-проект
# «Обоснование внедрения утилизационного платежа на электромобили в Республике Казахстан путем отмены нулевого коэффициента»

**Версия:** 1.0
**Статус:** Approved
**Дата:** 13.07.2026

## 1. Цель проекта
Разработать полностью автономный аналитический веб-проект, предназначенный для исследования, моделирования и обоснования внедрения утилизационного платежа на электромобили в Республике Казахстан посредством отмены действующего нулевого коэффициента.

Проект представляет собой локальный HTML-сайт, работающий без подключения к сети Интернет, без серверной части, без баз данных и без использования внешних API.

## 2. Основные принципы проекта
- Полностью локально
- Без установки
- Без регистрации
- Без интернета
- Без серверов
- Без баз данных

Запуск осуществляется открытием файла `index.html`.
